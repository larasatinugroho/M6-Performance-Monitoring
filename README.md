# M6-Performance-Monitoring
requested by the operational manager to represent M6 bucket performance with graphs

## Background overview
The current superset dashboard doesn't cover the progress of core KPI of M6 agents's : paid_up.
Hence, the operational manager request to represent M6 bucket performance through interactive graphs.
The goal is the manager able to see the track progress of each agents in bucket M6 without have to reorganize the raw data.
